# Answers

## 1. Did you use AI at any stage while working on this task? Why?

Yes, I used AI extensively during this task because this position emphasizes AI usage. Specifically, I used it for base project setup and overall scaffolding. The configuration process can be challenging due to rapid updates in tools like Prisma v7 and Tailwind v4, so manual configuration would have been significantly more time-consuming. Additionally, I used AI to build the basic UI, as I am not a designer, which could otherwise become a bottleneck. I also used AI to refactor certain parts following the approach I had already established.

## 2. What kind of problems or uncertainties AI helps you resolve during the process?

The initial setup can be confusing because most online search results point to outdated documentation. Using AI helped me overcome this uncertainty by providing up-to-date guidance. Additionally, building a UI without reference materials would be significantly more time-consuming than necessary. AI served as a reference guide for UI component design and implementation, accelerating the development process.
